# 🏦 Bank Management System (ATM Simulator)

A Java Swing-based desktop banking application with MySQL database integration.

---

## 💻 Technologies Used
- ☕ Java (Swing & AWT)
- 🗄️ MySQL Database
- 🔗 JDBC Connectivity
- 🖥️ NetBeans IDE

---

## ✨ Features
- 🏦 Account Opening
- 🔐 Login Authentication
- 💰 Deposit Money
- 💸 Cash Withdrawal
- ⚡ Fast Cash Option
- 📄 Mini Statement
- 📊 Balance Enquiry
- 🔑 PIN Change

---

## 🗄️ Database
MySQL database is used to store:
- Customer account details
- Transaction records

---


## 🚀 Purpose of Project
This project was built to understand:
- Java GUI development using Swing
- Event handling in Java
- JDBC database connectivity
- Real-world banking system logic
